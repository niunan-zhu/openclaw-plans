# 市场舆情监测方案

## 需求概述

- **目标**：每日自动抓取品牌、竞品、行业动态，整理为结构化报告
- **范围**：公众号、新闻资讯平台（百家号、头条号、搜狐号）、抖音、B站
- **推送**：邮件

---

## 方案一：免费方案（自建）

### 技术栈

| 组件 | 工具 | 成本 |
|------|------|------|
| 爬虫 | Playwright / Puppeteer | 0 |
| 存储 | SQLite | 0 |
| 调度 | OpenClaw Cron | 0 |
| 推送 | Nodemailer | 0 |

### 数据源与实现

| 平台 | 方案 | 难度 |
|------|------|------|
| 微信公众号 | 微信读书 RSS / Google Alert | 低 |
| 百家号 | 百度搜索 API（免费配额） | 低 |
| 头条号 | 今日头条搜索 | 中 |
| 搜狐号 | 搜狐搜索 | 低 |
| 抖音 | 抖音搜索（模拟请求） | 高 |
| B站 | B站开放平台 API | 低 |

### 优点
- 零成本
- 完全可控
- 数据存本地

### 缺点
- 抖音/B站需要模拟登录或付费接口
- 维护成本较高（反爬策略变化）
- 部分平台无法抓取

### 目录结构

```
scripts/
├── monitor/
│   ├── config/
│   │   ├── brand.json       # 品牌词
│   │   ├── competitor.json  # 竞品词
│   │   ├── benchmark.json   # 对标品牌
│   │   └── industry.json    # 行业词
│   ├── src/
│   │   ├── fetcher.js       # 抓取器
│   │   ├── storage.js      # 存储
│   │   └── reporter.js     # 报告生成
│   └── index.js             # 入口
```

## 方案二：付费方案（Saas）

### 推荐工具

| 平台 | 价格 | 说明 |
|------|------|------|
| 新榜 | ¥299/月起 | 公众号+抖音+B站 |
| 蝉妈妈 | ¥199/月起 | 抖音数据 |
| 5118 | ¥99/月起 | 全网营销数据 |
| 清博 | ¥500/月起 | 舆情监测 |

### 技术栈

| 组件 | 工具 | 成本 |
|------|------|------|
| 数据源 | 5118 API | ¥99/月 |
| 存储 | SQLite | 0 |
| 调度 | OpenClaw Cron | 0 |
| 推送 | Nodemailer | 0 |

### 优点
- 数据准确、全面
- 稳定更新
- 无反爬困扰

### 缺点
- 付费成本
- 数据依赖第三方
